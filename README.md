# UART
Написать на языке C следующие функции-драйверы нескольких устройств, входящих в состав системы на кристалле:
1)	Функция вывода строки с помощью контроллера UART с произвольным числом символов, последний символ маркируется как “\0”. Функция в качестве аргумента принимает указатель на эту строку
2)	Функция приема с помощью контроллера UART произвольного числа символов, определяемого аргументом функции и загрузку в ОЗУ по адресу, передаваемому в качестве аргумента.
Требуется написать предлагаемые функции на языке C, допускается использование ассемблерных вставок архитектуры MIPS-32 rev.6 либо RV32I на выбор студента.
