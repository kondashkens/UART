# UART
Написать на Cи функции-драйверы: 1)Ф-ция вывода строки с UART с произвольным числом символов, последний символ маркируется “\0”. Ф-ция в качестве аргумента принимает указатель на строку. 2)Ф-ция приема с UART произвольного числа символов, определяемого аргументом ф-ции и загрузку в ОЗУ по адресу, передаваемому в качестве аргумента.
